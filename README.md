🛠 Common Issues
1. App not loading or crashing
Try:
Closing and reopening the app
Updating to the latest version
Restarting your device
If the issue continues, email us with:
Device model
iOS/Android version
Screenshot or screen recording
2. Data not syncing
Make sure:
You are logged into the same account
You have a stable internet connection
Then pull down to refresh or restart the app.
3. Categories or features not working
If something feels broken:
Tell us exactly what you were doing
Send a screenshot if possible
We fix bugs quickly and push updates regularly.
